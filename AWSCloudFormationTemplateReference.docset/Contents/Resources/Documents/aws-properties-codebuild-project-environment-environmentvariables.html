AWS CodeBuild Project Environment EnvironmentVariables

EnvironmentVariables is a property of the AWS CodeBuild Project Environment property that specifies the name and value of an environment variable for an AWS CodeBuild project environment. When you use the environment to run a build, these variables are available for your builds to use.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

Copy
{
  "Name" : String,
  "Value" : String
}

YAML

Copy
Name: String
Value: String

Properties

Name

The name of an environment variable.

Required: Yes

Type: String

Value

The value of the environment variable.

Required: Yes

Type: String
