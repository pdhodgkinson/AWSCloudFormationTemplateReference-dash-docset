Amazon Cognito IdentityPoolRoleAttachment RoleMapping RulesConfiguration

RulesConfiguration is a subproperty of the AWS::Cognito::IdentityPoolRoleAttachment property that defines the rules to be used for mapping users to roles.

Syntax

JSON

Copy
{
  "Rules" : [ MappingRule, .. ]
}

YAML

Copy
Rules: 
  - MappingRule

Properties

Rules

A list of rules. You can specify up to 25 rules per identity provider.

Required: Yes

Type: List of Amazon Cognito IdentityPoolRoleAttachment MappingRule
