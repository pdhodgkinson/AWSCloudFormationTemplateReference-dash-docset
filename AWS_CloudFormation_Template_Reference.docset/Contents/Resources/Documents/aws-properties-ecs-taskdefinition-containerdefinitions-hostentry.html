Amazon Elastic Container Service TaskDefinition HostEntry

HostEntry is a property of the Amazon Elastic Container Service TaskDefinition ContainerDefinition property that specifies the hostnames and IP address entries to add to the Amazon Elastic Container Service (Amazon ECS) container's /etc/hosts file.

Syntax

JSON

{
  "Hostname" : String,
  "IpAddress" : String
}

YAML

Hostname: String
IpAddress: String

Properties

Hostname

The hostname to use in the /etc/hosts file.

Required: Yes

Type: String

IpAddress

The IP address to use in the /etc/hosts file.

Required: Yes

Type: String
