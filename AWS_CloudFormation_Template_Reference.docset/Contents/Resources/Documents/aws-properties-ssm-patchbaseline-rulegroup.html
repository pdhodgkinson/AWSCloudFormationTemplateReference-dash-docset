Amazon EC2 Systems Manager PatchBaseline RuleGroup

The RuleGroup property type specifies a set of rules that define the approval rules for an Amazon EC2 Systems Manager patch baseline.

RuleGroup is the property type for the ApprovalRules property of the AWS::SSM::PatchBaseline resource.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "PatchRules" : [ Rule, ... ]
}

YAML

PatchRules: 
  - Rule

Properties

PatchRules

The rules that make up the rule group.

Required: No

Type: List of SSM PatchBaseline Rule

Update requires: No interruption
