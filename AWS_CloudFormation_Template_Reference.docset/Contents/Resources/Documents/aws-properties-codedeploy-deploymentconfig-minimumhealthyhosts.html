CodeDeploy DeploymentConfig MinimumHealthyHosts

MinimumHealthyHosts is a property of the AWS::CodeDeploy::DeploymentConfig resource that defines how many instances must remain healthy during an CodeDeploy deployment.

Syntax

JSON

{
  "Type" : String,
  "Value" : Integer
}

YAML

Type: String
Value: Integer

Properties

Type

The type of count to use, such as an absolute value or a percentage of the total number of instances in the deployment. For valid values, see MinimumHealthyHosts in the AWS CodeDeploy API Reference.

Required: Yes

Type: String

Value

The minimum number of healthy instances.

Required: Yes

Type: Integer
