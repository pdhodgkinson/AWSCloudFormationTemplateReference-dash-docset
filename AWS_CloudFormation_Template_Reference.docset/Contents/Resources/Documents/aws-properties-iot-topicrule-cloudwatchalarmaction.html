AWS IoT TopicRule CloudwatchAlarmAction

CloudwatchAlarm is a property of the Actions property that describes an action that updates a CloudWatch alarm.

Syntax

JSON

Copy
{
  "AlarmName": String,
  "RoleArn": String,
  "StateReason": String,
  "StateValue": String
}

YAML

Copy
AlarmName: String
RoleArn: String
StateReason: String
StateValue: String

Properties

AlarmName

The CloudWatch alarm name.

Required: Yes

Type: String

RoleArn

The IAM role that allows access to the CloudWatch alarm.

Required: Yes

Type: String

StateReason

The reason for the change of the alarm state.

Required: Yes

Type: String

StateValue

The value of the alarm state.

Required: Yes

Type: String
