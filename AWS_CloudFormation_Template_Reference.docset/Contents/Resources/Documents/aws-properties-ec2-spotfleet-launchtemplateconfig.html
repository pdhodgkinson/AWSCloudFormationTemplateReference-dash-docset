Amazon Elastic Compute Cloud SpotFleet LaunchTemplateConfig

LaunchTemplateConfig is a property of the Amazon EC2 SpotFleet SpotFleetRequestConfigData property that describes a launch template and overrides.

Syntax

JSON

{
  "LaunchTemplateSpecification" : LaunchTemplateSpecification,
  "Overrides" : [ LaunchTemplateOverrides, ... ]
}

YAML

LaunchTemplateSpecification: LaunchTemplateSpecification
Overrides:
  - LaunchTemplateOverrides

Properties

LaunchTemplateSpecification

The launch template.

Required: Yes

Type: FleetLaunchTemplateSpecification

Update requires: No interruption

Overrides

Any parameters that you specify override the same parameters in the launch template.

Required: No

Type: List of LaunchTemplateOverrides

Update requires: No interruption
