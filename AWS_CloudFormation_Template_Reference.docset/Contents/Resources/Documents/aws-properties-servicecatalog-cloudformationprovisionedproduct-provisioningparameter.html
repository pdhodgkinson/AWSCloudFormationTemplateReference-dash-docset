AWS Service Catalog CloudFormationProvisionedProduct ProvisioningParameter

The ProvisioningParameter property type specifies a parameter for an AWS Service Catalog provisioned product.

ProvisioningParameter is a property of the AWS::ServiceCatalog::CloudFormationProvisionedProduct resource.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Value" : String,
  "Key" : String
}

YAML


Value: String
Key: String

Properties

Key

The parameter key.

Required: No

Type: String

Update requires: No interruption

Value

The parameter value.

Required: No

Type: String

Update requires: No interruption
