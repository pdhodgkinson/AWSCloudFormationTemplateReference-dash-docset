EC2 NetworkInterface Ipv6Addresses

Ipv6Addresses is a property of the AWS::EC2::NetworkInterface resource that specifies an IPv6 address to associate with the network interface.

Syntax

JSON

Copy
{
   "Ipv6Address" : String
}

YAML

Copy
Ipv6Address: String

Properties

Ipv6Address

The IPv6 address to associate with the network interface.

Required: Yes

Type: String
