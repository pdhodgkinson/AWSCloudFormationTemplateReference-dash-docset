AWS::ServiceCatalog::PortfolioShare

Shares the specified portfolio for AWS Service Catalog with the specified account. For more information, see CreatePortfolioShare in the AWS Service Catalog Developer Guide.

Topics

  • Syntax
  • Properties
  • Return Values

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Type" : "AWS::ServiceCatalog::PortfolioShare",
  "Properties" : {
    "AccountId" : String,
    "AcceptLanguage" : String,
    "PortfolioId" : String
  }
}

YAML

Type: "AWS::ServiceCatalog::PortfolioShare"
Properties:
  AccountId: String
  AcceptLanguage: String
  PortfolioId: String

Properties

AccountId

The AWS account ID.

Required: Yes

Type: String

Update requires: Replacement

AcceptLanguage

The language code.

Required: No

Type: String

Update requires: Replacement

PortfolioId

The portfolio identifier.

Required: Yes

Type: String

Update requires: Replacement

Return Values

Ref

When you pass the logical ID of an AWS::ServiceCatalog::PortfolioShare resource to the intrinsic Ref function, the function returns the identifier of the portfolio share.

For more information about using the Ref function, see Ref.
