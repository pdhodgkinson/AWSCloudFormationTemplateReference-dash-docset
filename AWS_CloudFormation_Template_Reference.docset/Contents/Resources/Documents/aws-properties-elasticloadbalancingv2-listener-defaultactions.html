Elastic Load Balancing Listener DefaultActions

DefaultActions is a property of the AWS::ElasticLoadBalancingV2::Listener resource that specifies the default actions the Elastic Load Balancing listener takes when handling incoming requests.

Syntax

JSON

Copy
{
  "TargetGroupArn" : String,
  "Type" : String
}

YAML

Copy
TargetGroupArn: String
Type: String

Properties

TargetGroupArn

The Amazon Resource Name (ARN) of the target group to which Elastic Load Balancing routes the traffic.

Required: Yes

Type: String

Type

The type of action. For valid values, see the Type contents for the Action data type in the Elastic Load Balancing API Reference version 2015-12-01.

Required: Yes

Type: String
