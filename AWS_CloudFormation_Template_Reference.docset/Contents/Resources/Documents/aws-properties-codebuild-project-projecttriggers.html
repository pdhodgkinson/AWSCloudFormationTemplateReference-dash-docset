AWS CodeBuild Project ProjectTriggers

ProjectTriggers is a property of the AWS::CodeBuild::Project resource that specifies the environment for an AWS CodeBuild project.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON 

{
  "Webhook" : Boolean
  "FilterGroups" : [[WebhookFilter, ...], ...]
}

YAML

Webhook: Boolean
FilterGroups: 
 - - WebhookFilter

Properties

Webhook

Specifies whether or not to begin automatically rebuilding the source code every time a code change is pushed to the repository.

Required: No

Type: Boolean

FilterGroups

A list of lists of WebhookFilter objects used to determine which webhook events are triggered. At least one WebhookFilter in the array must specify EVENT as its type.

Required: No

Type: A list of a list of AWS CodeBuild Project WebhookFilter objects.
