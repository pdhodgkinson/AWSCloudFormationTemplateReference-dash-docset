AWS::Cognito::UserPool SmsConfiguration

SmsConfiguration is a property of the AWS::Cognito::UserPool resource that defines the SMS configuration of an Amazon Cognito User Pool.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "ExternalId" : String,
  "SnsCallerArn" : String
}

YAML

  ExternalId: String
  SnsCallerArn: String

Properties

ExternalId

The external ID used in IAM role trust relationships.

For more information about using external IDs, see How to Use an External ID When Granting Access to Your AWS Resources to a Third Party in the AWS Identity and Access Management User Guide.

Required: No

Type: String

Update requires: No interruption

SnsCallerArn

The Amazon Resource Name (ARN) of the Amazon Simple Notification Service (SNS) caller.

Required: No

Type: String

Minimum: 20

Maximum: 2048

Pattern: arn:[\w+=/,.@-]+:[\w+=/,.@-]+:([\w+=/,.@-]*)?:[0-9]+:[\w+=/,.@-]+(:[\w+=/,.@-]+)?(:[\w+=/,.@-]+)?

Update requires: No interruption
