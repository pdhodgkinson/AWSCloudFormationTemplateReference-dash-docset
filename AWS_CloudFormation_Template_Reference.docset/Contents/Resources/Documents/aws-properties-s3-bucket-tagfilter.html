Amazon S3 Bucket TagFilter

The TagFilter property type specifies tags to use to identify a subset of objects for an Amazon S3 bucket.

The TagFilters property of the AWS::S3::Bucket property type contains a list of TagFilter property types.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Key" : String,
  "Value" : String
}

YAML

Key: String
Value: String

Properties

Key

The tag key.

Required: Yes

Type: String

Update requires: No interruption

Value

The tag value.

Required: Yes

Type: String

Update requires: No interruption
