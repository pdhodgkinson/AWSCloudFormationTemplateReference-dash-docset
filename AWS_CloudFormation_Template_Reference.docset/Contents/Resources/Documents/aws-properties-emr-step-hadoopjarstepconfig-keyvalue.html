Amazon EMR Step KeyValue

KeyValue is a property of the Amazon EMR Step HadoopJarStepConfig property that specifies key-value pairs, which are passed to a JAR file that Amazon EMR (Amazon EMR) executes.

Syntax

JSON

Copy
{
  "Key" : String,
  "Value" : String
}

YAML

Copy
Key: String
Value: String

Properties

Key

The unique identifier of a key-value pair.

Required: No

Type: String

Value

The value part of the identified key.

Required: No

Type: String
