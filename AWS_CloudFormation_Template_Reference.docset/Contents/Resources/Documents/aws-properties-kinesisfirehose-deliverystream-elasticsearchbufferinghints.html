Amazon Kinesis Firehose DeliveryStream ElasticsearchBufferingHints

The ElasticsearchBufferingHints property type specifies how Amazon Kinesis Firehose (Kinesis Firehose) buffers incoming data while delivering it to the destination. The first buffer condition that is satisfied triggers Kinesis Firehose to deliver the data.

ElasticsearchBufferingHints is the property type for the BufferingHints property of the Amazon Kinesis Firehose DeliveryStream ElasticsearchDestinationConfiguration property type.

Syntax

JSON

Copy
{
  "IntervalInSeconds" : Integer,
  "SizeInMBs" : Integer
}

YAML

Copy
IntervalInSeconds: Integer
SizeInMBs: Integer

Properties

IntervalInSeconds

The length of time, in seconds, that Kinesis Firehose buffers incoming data before delivering it to the destination. For valid values, see the IntervalInSeconds content for the BufferingHints data type in the Amazon Kinesis Firehose API Reference.

Required: Yes

Type: Integer

SizeInMBs

The size of the buffer, in MBs, that Kinesis Firehose uses for incoming data before delivering it to the destination. For valid values, see the SizeInMBs content for the BufferingHints data type in the Amazon Kinesis Firehose API Reference.

Required: Yes

Type: Integer
