CodePipeline Pipeline ArtifactStore

ArtifactStore is a property of the AWS::CodePipeline::Pipeline resource that defines the S3 location where CodePipeline stores pipeline artifacts.

Syntax

JSON

{
  "EncryptionKey" : EncryptionKey,
  "Location" : String,
  "Type" : String
}

YAML

EncryptionKey: EncryptionKey
Location: String
Type: String

Properties

EncryptionKey

The encryption key CodePipeline uses to encrypt the data in the artifact store, such as an AWS Key Management Service (AWS KMS) key. If you don't specify a key, CodePipeline uses the default key for Amazon Simple Storage Service (Amazon S3).

Required: No

Type: EncryptionKey

Location

The name of the S3 bucket where CodePipeline stores artifacts for a pipeline.

Required: Yes

Type: String

Type

The type of the artifact store, such as Amazon S3. For valid values, see ArtifactStore in the AWS CodePipeline API Reference.

Required: Yes

Type: String
