AWS OpsWorks Stack ElasticIp

ElasticIps is a property of the AWS::OpsWorks::Stack resource that registers an Elastic IP address with an AWS OpsWorks stack.

Syntax

JSON

{
  "Ip" : String,
  "Name" : String
}

YAML

Ip: String
Name: String

Properties

Ip

The Elastic IP address.

Required: Yes

Type: String

Name

A name for the Elastic IP address.

Required: No

Type: String
