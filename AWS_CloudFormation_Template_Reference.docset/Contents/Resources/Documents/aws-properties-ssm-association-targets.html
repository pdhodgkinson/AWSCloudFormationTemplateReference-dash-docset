AWS Systems Manager Association Targets

Targets is a property of the AWS::SSM::Association resource that specifies the targets for an SSM document in Systems Manager.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Key" : String,
  "Values" : [ String, ... ]
}

YAML

Key: String
Values:
  - String

Properties

Key

The name of the criteria that EC2 instances must meet. For valid keys, see the Target data type in the AWS Systems Manager API Reference.

Required: Yes

Type: String

Values

The value of the criteria. Systems Manager runs targeted commands on EC2 instances that match the criteria. For more information, see the Target data type in the AWS Systems Manager API Reference.

Required: Yes

Type: List of String values
