Amazon EMR Cluster Application

Application is a property of the AWS::EMR::Cluster resource that adds an Amazon EMR (Amazon EMR) application bundle or third-party software to an Amazon EMR cluster.

Syntax

JSON

Copy
{
  "AdditionalInfo" : { String:String, ... },
  "Args" : [ String, ... ],
  "Name" : String,
  "Version" : String
}

YAML

Copy
AdditionalInfo:
  String: String
Args:
  - String
Name: String
Version: String

Properties

AdditionalInfo

Metadata about third-party applications that third-party vendors use for testing purposes.

Required: No

Type: String-to-string map

Args

Arguments that Amazon EMR passes to the application.

Required: No

Type: List of String values

Name

The name of the application to add to your cluster, such as Hadoop or Hive. For valid values, see the Applications parameter in the Amazon EMR API Reference.

Required: No

Type: String

Version

The version of the application.

Required: No

Type: String
