Amazon S3 Bucket SseKmsEncryptedObjects

The SseKmsEncryptedObjects property type specifies the status of whether Amazon S3 replicates objects created with server-side encryption using an AWS KMS-managed key.

SseKmsEncryptedObjects is a property of the AWS::S3::Bucket resource.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Status" : String
}

YAML

Status: String

Properties

Status

Specifies whether Amazon S3 replicates objects created with server-side encryption using an AWS KMS-managed key. Valid values include Enabled and Disabled.

Required: Yes

Type: String

Update requires: No interruption
