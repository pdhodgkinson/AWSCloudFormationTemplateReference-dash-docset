AWS CodeDeploy DeploymentGroup Alarm

The Alarm property type specifies a CloudWatch alarm to use for an AWS CodeDeploy deployment group. The Alarm property of the AWS CodeDeploy DeploymentGroup AlarmConfiguration property contains a list of Alarm property types.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

Copy
{
  "Name" : String
}

YAML

Copy
Name: String

Properties

Name

The name of the alarm. For more information, see Alarm in the AWS CodeDeploy API Reference.

Required: No

Type: String

Update requires: No interruption
