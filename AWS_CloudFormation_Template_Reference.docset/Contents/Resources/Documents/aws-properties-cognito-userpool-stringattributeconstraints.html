Amazon Cognito UserPool StringAttributeConstraints

The StringAttributeConstraints property type defines the string attribute constraints of an Amazon Cognito User Pool. StringAttributeConstraints is a subproperty of the Amazon Cognito UserPool SchemaAttribute property type.

Syntax

JSON

{
  "MaxLength" : String,
  "MinLength" : String
}

YAML

MaxLength: String
MinLength: String

Properties

MaxLength

The maximum value of an attribute that is of the string data type.

Type: String

Required: No

MinLength

The minimum value of an attribute that is of the string data type.

Type: String

Required: No
