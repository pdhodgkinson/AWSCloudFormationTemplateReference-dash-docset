Amazon Kinesis Data Analytics Application InputParallelism

The InputParallelism property type specifies the number of in-application streams to create for a given streaming source in an Amazon Kinesis Data Analytics application.

InputParallelism is a property of the Kinesis Data Analytics Application Input property type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Count" : Integer
}

YAML


  Count: Integer

Properties

Count

The number of in-application streams to create.

Required: No

Type: Integer

Update requires: No interruption
