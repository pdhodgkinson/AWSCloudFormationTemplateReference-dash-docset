Amazon Cognito UserPool InviteMessageTemplate

InviteMessageTemplate is a property of the AWS::Cognito::UserPool resource that defines the email invitation message template of an Amazon Cognito User Pool.

Syntax

JSON

Copy
{
  "EmailMessage" : String,
  "EmailSubject" : String,
  "SMSMessage" : String
}

YAML

Copy
EmailMessage: String
EmailSubject: String
SMSMessage: String

Properties

EmailMessage

The message template for email messages.

Type: String

Required: No

EmailSubject

The subject line for email messages.

Type: String

Required: No

SMSMessage

The message template for SMS messages.

Type: String

Required: No
