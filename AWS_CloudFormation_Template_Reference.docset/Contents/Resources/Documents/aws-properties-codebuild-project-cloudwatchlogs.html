AWS CodeBuild Project CloudWatchLogs

CloudWatchLogs is a property of the AWS::CodeBuild::Project resource that specifies settings for CloudWatch Logs generated by an AWS CodeBuild build.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Status" : String,
  "GroupName" : String,
  "StreamName" : String
}

YAML

Status:  String
GroupName:  String
StreamName:  String

Properties

Status

The current status of the CloudWatch Logs for a build project. Valid values are:

  • ENABLED: CloudWatch Logs are enabled for this build project.

  • DISABLED: CloudWatch Logs are not enabled for this build project.

Required: Yes

Type: String

Update requires: No interruption

GroupName

The group name of the CloudWatch Logs.

Required: No

Type: String

Update requires: No interruption

StreamName

The prefix of the stream name of the CloudWatch Logs.

Required: No

Type: String

Update requires: No interruption
