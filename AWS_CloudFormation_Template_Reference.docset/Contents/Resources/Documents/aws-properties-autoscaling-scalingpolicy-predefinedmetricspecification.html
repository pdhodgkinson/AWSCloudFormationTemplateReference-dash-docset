Amazon EC2 Auto Scaling ScalingPolicy PredefinedMetricSpecification

The PredefinedMetricSpecification property configures a predefined metric for a target tracking policy in Amazon EC2 Auto Scaling. PredefinedMetricSpecification is a subproperty of the Amazon EC2 Auto Scaling ScalingPolicy TargetTrackingConfiguration property.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "PredefinedMetricType" : String,
  "ResourceLabel" : String
}

YAML

PredefinedMetricType: String
ResourceLabel: String

Properties

For more information about each property, including constraints and valid values, see PredefinedMetricSpecification in the Amazon EC2 Auto Scaling API Reference.

PredefinedMetricType

The metric type.

Required: Yes

Type: String

Update requires: No interruption

ResourceLabel

Identifies the resource associated with the metric type.

Required: No

Type: String

Update requires: No interruption
