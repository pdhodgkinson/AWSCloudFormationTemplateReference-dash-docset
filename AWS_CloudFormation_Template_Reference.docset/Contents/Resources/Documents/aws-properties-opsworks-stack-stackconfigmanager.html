AWS OpsWorks StackConfigurationManager Type

Describes the stack configuration manager for the AWS::OpsWorks::Stack resource type. For more information, see StackConfigurationManager in the AWS OpsWorks Stacks API Reference.

Syntax

JSON

Copy
{
  "Name" : String,
  "Version" : String
}

YAML

Copy
Name: String
Version: String

Properties

Name

The name of the configuration manager.

Required: No

Type: String

Version

The Chef version.

Required: No

Type: String
