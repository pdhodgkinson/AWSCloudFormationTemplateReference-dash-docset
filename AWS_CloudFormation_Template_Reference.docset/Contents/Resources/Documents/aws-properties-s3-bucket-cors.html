Amazon S3 Bucket CorsConfiguration

Describes the cross-origin access configuration for objects in an AWS::S3::Bucket resource.

Syntax

JSON

Copy
{
  "CorsRules" : [ CorsRules, ... ]
}

YAML

Copy
CorsRules:
  - CorsRules

Properties

CorsRules

A set of origins and methods that you allow.

Required: Yes

Type: Amazon S3 Bucket CorsRule
