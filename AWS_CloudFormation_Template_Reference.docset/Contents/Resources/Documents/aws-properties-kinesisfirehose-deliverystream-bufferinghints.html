Amazon Kinesis Firehose DeliveryStream BufferingHints

The BufferingHints property type specifies how Amazon Kinesis Firehose (Kinesis Firehose) buffers incoming data before delivering it to the destination. The first buffer condition that is satisfied triggers Kinesis Firehose to deliver the data.

BufferingHints is a property of the Amazon Kinesis Firehose DeliveryStream ExtendedS3DestinationConfiguration and Amazon Kinesis Firehose DeliveryStream S3DestinationConfiguration property types.

Syntax

JSON

Copy
{
  "IntervalInSeconds" : Integer,
  "SizeInMBs" : Integer
}

YAML

Copy
IntervalInSeconds: Integer
SizeInMBs: Integer

Properties

IntervalInSeconds

The length of time, in seconds, that Kinesis Firehose buffers incoming data before delivering it to the destination. For valid values, see the IntervalInSeconds content for the BufferingHints data type in the Amazon Kinesis Firehose API Reference.

Required: Yes

Type: Integer

SizeInMBs

The size of the buffer, in MBs, that Kinesis Firehose uses for incoming data before delivering it to the destination. For valid values, see the SizeInMBs content for the BufferingHints data type in the Amazon Kinesis Firehose API Reference.

Required: Yes

Type: Integer
